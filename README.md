# Samples
This is a small project made for making chiptune style music.

It has the already made urls which you need for use in ultrabox.

I also have the samples in as actual downloadable audio files if you dont want to use ultrabox.
their in the audio files branch.

This project is specifically and only designed for samples on ultrabox any changes to the website or anything related is simply not my problem.

in order to use the samples I'll be putting in this project you'll get the main file which I haven't made yet and it'll be explicitly labeled for easy navigation.

you just copy any URL of your choosing go to ultrabox select edit scroll to the bottom select "add custom samples" after that press "add sample" or if your pasting multiple "add multiple files" press ok and wait for it to load.
